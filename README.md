# Placement Tracker System

## 📌 Overview
The Placement Tracker System is a cloud-based application developed using Zoho Creator, a low-code development platform, to help students efficiently manage and track their job and internship applications.

This system digitizes the placement tracking process by allowing students to store company details, upload resumes, track interview rounds, monitor application status, and view placement analytics through an interactive dashboard.

The application improves organization, reduces manual tracking using spreadsheets, and provides real-time insights into placement progress.

---

## 🎯 Objective
The primary goal of this system is to:

- Eliminate manual tracking of placement applications
- Provide a centralized platform to manage job and internship applications
- Track interview rounds and application status
- Store company and job details efficiently
- Provide dashboard analytics for placement progress
- Improve student productivity and placement organization

---

## 🚀 Key Features

### 👨‍🎓 Student Module
- Add and manage job applications
- Upload resumes for each application
- Track application status (Applied, Interview, Selected, Rejected)
- Monitor interview schedules
- View placement progress dashboard

### 🏢 Company Module
- Add company details
- Store job roles and descriptions
- Track salary or stipend information
- Record application deadlines

### 📅 Interview Tracking Module
- Add interview dates
- Track interview rounds (Technical, HR, Aptitude)
- Update interview results
- Store interview notes

### 📊 Dashboard Module
- View total applications
- View total companies applied
- Track selected and rejected applications
- View application status distribution
